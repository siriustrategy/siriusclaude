# 🌟 Squad Sirius Proposals — Comandos Instalados

Squad de propostas comerciais para a Sirius Strategy.

## Agentes disponíveis

| Slash Command | Agente | Função |
|--------------|--------|--------|
| `/squad-sirius-proposals:agents:sirius-proposals-chief` | Chief | Inicia e orquestra toda a proposta |

## Como usar

```
/squad-sirius-proposals:agents:sirius-proposals-chief
```

Depois de ativar, diga `*nova-proposta` para começar.

## O que o squad entrega

Para cada cliente:
- Deck Google Slides (branding Sirius + logo do cliente)
- Script de apresentação slide a slide
- One-pager executivo
- Planilha de ROI
- Cronograma de implementação
- Pré-contrato
- Email de follow-up
- Mapeamento de objeções + respostas
- Case study por setor
