## 🏢 SOBRE A SIRIUS STRATEGY

**Especialidade:** IA, Automações, Agentes, Dashboards, Sites e Soluções Omnichannel
**Diferencial:** Hub de soluções integradas que se comunicam — cada solução complementa a outra
**Modelo:** Implementação única + MRR (fee mensal para manutenção e evolução)
**Stack principal:** Claude Code + AIOS, Supabase, Node.js/Python, WhatsApp API, Vercel/VPS

## 🛠️ CATÁLOGO DE SERVIÇOS

### 1. SITES & LANDING PAGES
**Descrição:** Sites modernos, landing pages de alta conversão, otimizados para SEO e performance.
**Tecnologias:** Next.js, Vercel, Supabase, SEO Google + AI
**Entregáveis:** Site completo, painel de edição, domínio conectado, analytics
**Combinações naturais:** + Agente de qualificação de leads, + CRM dashboard, + Checkout

---

### 2. DASHBOARDS & BI
**Descrição:** Painéis de dados em tempo real para tomada de decisão — vendas, operações, financeiro, CRM.
**Tecnologias:** Supabase, Power BI, Looker, SQL, visualizações customizadas
**Entregáveis:** Dashboard funcional, fontes de dados conectadas, relatórios automáticos
**Combinações naturais:** + CRM customizado, + Agente de relatórios, + Automação de dados

---

### 3. AGENTES DE IA
**Descrição:** Agentes autônomos que executam tarefas sem intervenção humana — atendimento, cobrança, qualificação de leads, suporte, follow-up.
**Tecnologias:** Claude/OpenAI API, WhatsApp API (Z-API/Twilio), integrações via webhook
**Entregáveis:** Agente configurado, integrado ao canal, testado e documentado
**Exemplos reais:**
  - Agente de pré-qualificação de leads (WhatsApp → CRM)
  - Agente de cobrança/inadimplência (detecta → comunica → negocia)
  - Agente de atendimento (triagem → resposta → escalonamento)
  - Agente de onboarding (recebe novo cliente → guia → coleta dados)
**Combinações naturais:** + CRM dashboard, + Site com formulário, + Automações

---

### 4. AUTOMAÇÕES & WORKFLOWS
**Descrição:** Sequências de tarefas que rodam sozinhas, sem intervenção manual a cada passo.
**Tecnologias:** Node.js/Python, webhooks, APIs, integrações
**Entregáveis:** Workflow documentado, testado, monitorado
**Exemplos reais:**
  - Workflow de inadimplência: detecta → WhatsApp → email → escala → fecha
  - Workflow de onboarding: lead → valida → cria conta → boas-vindas
  - Workflow de relatório: coleta dados → processa → gera → envia
  - Workflow de nutrição: lead entra → sequência personalizada → qualifica
**Combinações naturais:** + Agentes de IA, + CRM, + Disparos WhatsApp

---

### 5. CRM CUSTOMIZADO
**Descrição:** Dashboard de CRM feito sob medida para o negócio — sem pagar por ferramenta cara, sem adaptação.
**Tecnologias:** Supabase, Next.js/React, autenticação, relatórios
**Entregáveis:** CRM funcional, histórico de clientes, funil de vendas, integração com canais
**Combinações naturais:** + Agente de leads (WhatsApp), + Site, + Automações

---

### 6. CHECKOUT & PAGAMENTOS
**Descrição:** Páginas de checkout integradas, com gateway de pagamento, identificação do cliente e atualização automática de status.
**Tecnologias:** Stripe/Asaas/PagSeguro, Supabase, webhooks
**Entregáveis:** Página de checkout, gateway configurado, integração com CRM
**Combinações naturais:** + Site, + CRM, + Agente de cobrança

---

### 7. INTEGRAÇÃO OMNICHANNEL
**Descrição:** Conexão de todos os canais (site, WhatsApp, Instagram, email) em um hub centralizado.
**Tecnologias:** Z-API, Twilio, webhooks, Supabase, APIs
**Entregáveis:** Canais conectados, histórico unificado, regras de roteamento
**Combinações naturais:** + Agentes de IA, + CRM, + Automações

---

### 8. SQUADS DE AGENTES CUSTOMIZADOS (AIOS)
**Descrição:** Equipes de agentes IA especializados que trabalham juntos para processos complexos.
**Tecnologias:** Claude Code + AIOS, APIs, integrações
**Exemplos:**
  - Squad de Vendas: qualificação → nurturing → follow-up → fechamento
  - Squad de Suporte: triagem → resposta → resolução → satisfação
  - Squad de Marketing: conteúdo → publicação → análise → otimização
**Combinações naturais:** + CRM, + WhatsApp, + Dashboard

## 🔗 COMBINAÇÕES DE HUB (EXEMPLOS PODEROSOS)

### Hub de Aquisição de Clientes
```
Site (landing page) → Agente de pré-qualificação WhatsApp → Relatório no CRM → Follow-up automático
```
**Por que é poderoso:** O lead chega pelo site, é qualificado pelo agente, o time recebe um relatório completo no CRM já com todas as informações, e um follow-up automático mantém o lead quente.

### Hub de Retenção / Cobrança
```
CRM detecta vencimento → Agente dispara WhatsApp → Link de Checkout → CRM atualiza status
```
**Por que é poderoso:** Elimina call center manual, escala sem custo adicional de equipe.

### Hub de Operações
```
Dashboard de dados → Relatórios automáticos → Alertas via WhatsApp → Ação automática
```
**Por que é poderoso:** Gestão baseada em dados em tempo real, sem planilhas manuais.

### Hub Completo (máximo valor MRR)
```
Site + CRM + Agente WhatsApp + Checkout + Dashboard + Automações
= Plataforma completa do negócio
```
