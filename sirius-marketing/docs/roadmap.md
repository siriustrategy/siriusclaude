## MVP Mínimo para Primeiro Usuário

Para uma primeira versão funcional e impressionante, o mínimo é:
1. Login + 1 marca configurada (Brand Onboarding)
2. Viral Feed funcionando (pelo menos manual, não automático)
3. Geração de imagem com 1 engine (Flux)
4. Brand Police básico
5. Copy Engine
6. Sistema de créditos

**Estimativa MVP:** 6-8 semanas de desenvolvimento focado.
