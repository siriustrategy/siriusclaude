# Sirius-Marketing

Creative AI Studio para profissionais de marketing que querem criar conteúdo estratégico,
consistente e viral — orquestrado por agentes de IA especializados.

**Parte do ecossistema Sirius** | Integrado com Sirius Academy

---

## O que é

Uma plataforma que combina:
- Inteligência viral (o que está explodindo nas redes + por que)
- Geração de conteúdo com IA alinhada à identidade de marca
- Análise de performance automática das campanhas
- Publicação automática no TikTok e Meta
- Tudo isso dentro de um workflow estruturado com agentes especializados

## Squads

| Squad | Módulo | Agentes |

| squad-brand-guardian | Brand Hub | brand-analyst, brand-voice-trainer, brand-police |

| squad-creative-studio | Creative Studio | prompt-engineer, image-director, character-designer |
| squad-content-factory | Content Factory | copywriter, storyboard-director, calendar-strategist |
| squad-ideas-lab | Ideas Lab | idea-generator, seo-specialist, competitor-analyst |
| squad-analytics | Analytics | social-scraper, campaign-monitor, insights-analyst |
| squad-distribution | Distribution | scheduler, publisher, n8n-connector |
| squad-lovable-architect | Lovable Architect | requirements-interviewer, prompt-architect |
