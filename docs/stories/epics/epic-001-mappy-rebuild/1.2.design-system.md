## Story

**As a** designer do projeto Mappy,
**I want** um design system de componentes base sem AI-look,
**so that** todas as páginas tenham identidade visual coerente, premium e artesanal.

## Acceptance Criteria

1. Componente `Button` com variantes: primary, outline, ghost — com hover/focus artesanal
2. Componente `Badge` com variante de cor
3. Componente `Card` com hover sutil — sem shadow genérico
4. Componente `SectionHeader` (badge + headline + subtítulo)
5. Componente `Nav` sticky com logo, links, WhatsApp "PLANTÃO 24h" e CTA
6. Componente `Footer` com 4 colunas + CNPJ
7. Componente `WhatsAppButton` flutuante com mensagem pré-definida
8. Nenhum componente usa gradiente genérico roxo-azul
9. Todos os componentes são responsivos (mobile-first)
10. Animações usam CSS transitions artesanais OU Framer Motion — nunca `transition-all`

## Dev Notes

**Anti-patterns a evitar (referência: steve-schoger):**
- `transition-all` — usar propriedades específicas
- `shadow-lg` genérico — criar shadow custom com cor da marca
- `bg-gradient-to-r from-purple-500 to-blue-500` — proibido
- Ícones Heroicons sem adaptação
- `rounded-lg` em tudo — usar valores específicos por contexto

**Referência visual: memorialmais.com.br**
- Hero: foto real com overlay escuro gradiente LEFT→RIGHT
- Cards dos pilares: ícone outline + padding 32px + borda sutil
- Botões: radius moderado, não pill shape
- Nav: flat, sem shadow pesada, com contato visível
