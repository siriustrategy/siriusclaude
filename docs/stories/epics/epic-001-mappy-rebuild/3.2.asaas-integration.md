## Story

**As a** visitante da LP B2C,
**I want** poder assinar o plano Mappy diretamente online com cartão ou PIX/boleto,
**so that** eu me torne um assinante ativo sem precisar falar com ninguém.

## Acceptance Criteria

1. Botão "Assinar Agora" (cartão) abre checkout Asaas (link/redirect ou embed)
2. Botão "Assinar Agora" (PIX/boleto) abre checkout Asaas com método correto
3. Endpoint `/api/asaas/checkout` cria assinatura no Asaas e retorna link de pagamento
4. Endpoint `/api/asaas/webhook` recebe eventos do Asaas e atualiza Supabase
5. Eventos do webhook: `PAYMENT_CONFIRMED`, `PAYMENT_OVERDUE`, `SUBSCRIPTION_CANCELLED`
6. Assinante confirmado → salva em `asaas_subscribers` com status `ACTIVE`
7. Assinante inadimplente → status `OVERDUE`
8. Cancelamento → status `CANCELLED`
9. Webhook valida `asaas-access-token` header para segurança
10. Em sandbox: fluxo completo funciona sem erros

## Dev Notes

**Asaas API base URL:**
- Sandbox: `https://sandbox.asaas.com/api/v3`
- Produção: `https://api.asaas.com/v3`

**Criar customer:**
```ts
const customer = await asaasRequest('POST', '/customers', {
  name: nome,
  email,
  cpfCnpj: cpf,
  mobilePhone: telefone
})
```

**Criar assinatura:**
```ts
const subscription = await asaasRequest('POST', '/subscriptions', {
  customer: customer.id,
  billingType: planId === 'cartao' ? 'CREDIT_CARD' : 'BOLETO', // PIX também disponível
  value: planId === 'cartao' ? 139.90 : 149.90,
  nextDueDate: format(new Date(), 'yyyy-MM-dd'),
  cycle: 'MONTHLY',
  description: 'Mappy — Plano de Teleatendimento'
})
```

**Webhook handler:**
```ts
// Verificar autenticidade
const token = request.headers.get('asaas-access-token')
if (token !== process.env.ASAAS_WEBHOOK_TOKEN) {
  return Response.json({ error: 'Unauthorized' }, { status: 401 })
}

// Processar evento
const { event, payment, subscription } = await request.json()

if (event === 'PAYMENT_CONFIRMED') {
  await supabase.from('asaas_subscribers')
    .upsert({ asaas_subscription_id: subscription.id, status: 'ACTIVE', ... })
}
```

**Variáveis de ambiente necessárias:**
```
ASAAS_API_KEY=          # Chave da API Asaas (sandbox ou prod)
ASAAS_WEBHOOK_TOKEN=    # Token para validar webhooks
ASAAS_ENV=sandbox       # 'sandbox' ou 'production'
```

**Checkout flow:**
1. Usuário clica "Assinar Agora"
2. Modal abre pedindo: nome, email, CPF, telefone
3. POST `/api/asaas/checkout` → cria customer + subscription no Asaas
4. Response: `{ checkoutUrl: 'https://...' }`
5. Redirect para checkoutUrl (página de pagamento do Asaas)
6. Asaas processa pagamento
7. Asaas envia webhook → `/api/asaas/webhook`
8. Webhook salva assinante no Supabase
