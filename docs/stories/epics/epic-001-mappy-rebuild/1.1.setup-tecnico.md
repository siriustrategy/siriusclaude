## Story

**As a** desenvolvedor do projeto Mappy,
**I want** ter o projeto Next.js 14 configurado com Supabase, Vercel e GitHub,
**so that** todas as stories seguintes possam ser implementadas sobre uma base sólida e automaticamente deployada.

## Acceptance Criteria

1. Repositório GitHub `mappy-site` criado (privado)
2. Next.js 14 inicializado com App Router + TypeScript strict + Tailwind CSS
3. Framer Motion instalado e funcionando
4. Contentlayer + MDX instalado para o blog
5. Variáveis de ambiente configuradas (`.env.local` + `.env.example`)
6. Supabase: projeto criado + 4 tabelas criadas via SQL
7. Supabase client configurado no projeto (`lib/supabase.ts`)
8. Vercel: projeto conectado ao GitHub com deploy automático
9. Preview URL funcionando após push na main
10. `npm run build` passa sem erros
