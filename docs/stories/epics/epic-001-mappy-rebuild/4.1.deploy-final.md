## Story

**As a** equipe Mappy,
**I want** o site novo no ar em mappy.com.br com deploy automático,
**so that** qualquer push na main atualize o site automaticamente.

## Acceptance Criteria

1. Domínio mappy.com.br apontando para Vercel
2. HTTPS automático configurado
3. Deploy automático: push na `main` → Vercel deploya em < 2 minutos
4. Preview deployments para branches (cada PR tem preview URL)
5. Variáveis de ambiente de produção configuradas no Vercel
6. `npm run build` passa sem warnings críticos
7. Lighthouse score: Performance ≥ 80, SEO ≥ 90, Accessibility ≥ 80
8. Site antigo (Lovable) removido ou redirecionado
9. Sitemap.xml gerado e acessível em /sitemap.xml
10. robots.txt configurado

## Dev Notes

**Sitemap Next.js 14:**
```ts
// src/app/sitemap.ts
export default function sitemap(): MetadataRoute.Sitemap {
  return [
    { url: 'https://mappy.com.br', lastModified: new Date() },
    { url: 'https://mappy.com.br/white-label', lastModified: new Date() },
    { url: 'https://mappy.com.br/terapia-online', lastModified: new Date() },
    { url: 'https://mappy.com.br/blog', lastModified: new Date() },
    { url: 'https://mappy.com.br/contato', lastModified: new Date() },
    { url: 'https://mappy.com.br/empresas', lastModified: new Date() },
    { url: 'https://mappy.com.br/assinar', lastModified: new Date() },
    // posts do blog dinamicamente
    ...allPosts.map(post => ({
      url: `https://mappy.com.br/blog/${post.slug}`,
      lastModified: new Date(post.date)
    }))
  ]
}
```
