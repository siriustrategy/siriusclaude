## Story

**As a** visitante do site mappy.com.br,
**I want** ver um hero impactante, os serviços disponíveis e como funciona a plataforma,
**so that** eu entenda imediatamente o que é a Mappy e queira contratar.

## Acceptance Criteria

1. Hero: foto real (não mockup de chat) + overlay gradiente direcional + texto esquerda
2. Badge "White-Label · Implementação em 5 dias" acima do headline
3. Headline: "Sua plataforma de teleatendimento com a sua marca" — bold heavy
4. 2 CTAs opostos: "Montar Plano" → /comecar | "Acessar Sistema" → app.mappy.com.br
5. 3 métricas abaixo dos CTAs: "5 dias | +50k atendimentos | NPS 92,4%"
6. Seção Serviços: 6 cards com ícone outline — sem grid genérico uniforme
7. Seção escura (fundo #0A1628) com prova social numérica — break de ritmo
8. Seção "Como Funciona": 4 etapas numeradas com animação de entrada (Framer Motion)
9. Todas as seções têm animação de entrada ao scroll (Framer Motion `whileInView`)
10. Página totalmente responsiva (mobile-first)

## Dev Notes

**Hero overlay CSS (não usar bg-black/50 genérico):**
```css
.hero-overlay {
  background: linear-gradient(
    to right,
    rgba(10, 22, 40, 0.85) 0%,
    rgba(10, 22, 40, 0.50) 50%,
    rgba(10, 22, 40, 0.20) 100%
  );
}
```

**Framer Motion padrão para seções:**
```tsx
<motion.div
  initial={{ opacity: 0, y: 24 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true }}
  transition={{ duration: 0.5, ease: [0.22, 1, 0.36, 1] }}
>
```

**Stagger para cards:**
```tsx
transition={{ duration: 0.4, delay: index * 0.08 }}
```

- Badge discreto com borda sutil
- 2 CTAs: sólido (primário) + outline (secundário) — lado a lado
- Números grandes em seção escura como break de ritmo
