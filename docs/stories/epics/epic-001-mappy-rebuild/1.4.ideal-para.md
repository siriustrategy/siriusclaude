## Story

**As a** visitante do site,
**I want** ver quais tipos de empresa se beneficiam da Mappy com conteúdo dinâmico por segmento,
**so that** eu me identifique como cliente potencial e tome a ação certa.

## Acceptance Criteria

1. 6 tabs de segmento: Telecom, Clubes, SaaS RH, Seguradoras, Sindicatos, Cooperativas
2. Ao clicar no tab: conteúdo muda com animação fade+slide (Framer Motion AnimatePresence)
3. Tab ativo: fundo #35B0FF, texto branco, scale(1.05)
4. Dots indicadores: expandem de círculo para pill no ativo
5. Conteúdo dinâmico: título, descrição, 3 benefícios, métricas (5 dias / 100% white-label)
6. CTA dinâmico por segmento: "Quero para minha telecom", "Quero para meu clube", etc.
7. CTA secundário: "Agendar call" (link WhatsApp com mensagem específica)
8. Benefícios aparecem com stagger animation (0.1s entre cada)
9. Responsivo: tabs em wrap no mobile (2-3 por linha)
10. Componente reutilizável (será reusado na LP B2B)

**AnimatePresence pattern:**
```tsx
<AnimatePresence mode="wait">
  <motion.div
    key={selected}
    initial={{ opacity: 0, y: 20 }}
    animate={{ opacity: 1, y: 0 }}
    exit={{ opacity: 0, y: -20 }}
    transition={{ duration: 0.3, ease: [0.22, 1, 0.36, 1] }}
  >
    {/* conteúdo dinâmico */}
  </motion.div>
</AnimatePresence>
```
